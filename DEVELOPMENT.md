## Development

The code is [Kotlin](https://kotlinlang.org/) so, at the very least, Java (JDK21) needs to be available on your dev machine.

###### Downloads / Installation:

* [Adoptium JDK](https://adoptium.net/)
* [Kotlin](https://kotlinlang.org/)

### Getting Started

1) Clone or fork the repo
2) Run `./gradlew qa`
